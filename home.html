Javascript principal linguagem para desenvolvimento web

Minha relação com o Javascript começou desde que comecei a fazer aplicação web. Já tenho mais de oito anos trabalhando com a linguagem, que ao meu ver, é a mais importante para desenvolvimento para aplicações web. Os frameworks mais usados para desenvolvimento Web como Angular e React Native são feitos em Javascript, o angular ainda contém o TypeScript, que também é feito com Javascript. Por isso, para mim tal importancia do Javascript. Em breve irei fazer projetos usando os frameworks citados acima. Aguardem.

Node.js a multiplataforma do momento

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web Como um ambiente de execução JavaScript assíncrono orientado a eventos, o Node.js é projetado para desenvolvimento de aplicações escaláveis de rede. Hoje em dia é muito usado para desenvolvimento de microserviços, e é ai aonde entra minha experiência. Nos últimos quatro anos, eu atuei também em desenvolvimento de microserviços e api rest. Minhas proximas aplicações iram ter integração com uma api em node.js.
